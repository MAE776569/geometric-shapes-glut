# Drawing geometric shapes using GLUT

**_The OpenGL Utility Toolkit (GLUT) is a library of utilities for OpenGL programs._**

This program draws some geometric shapes:

- pentagon
- hexagon
- heptagon
- octagon
- nonagon
- decagon
